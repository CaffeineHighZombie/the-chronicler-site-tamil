    SQL சர்வருக்காக யாராவது தயவுசெய்து சிந்தியுங்கள்! – ஒரு டெக் சப்போர்ட் கதை

    "இது சர்வர் இல்லைங்க, இது வேற மாதிரி வேலை!" – இது போல வார்த்தைகள் எத்தனை டெக் சப்போர்ட் குரல் ரெக்கார்டிங்கில் கேட்டிருப்போம் தெரியுமா? தொழில்நுட்ப உலகத்தில், குறிப்பாக SQL, அனுமதிகள், வெப் பக்கம், போன்றவற்றோடு வேலை செய்யும் போது, நம்ம தமிழ்நாட்டு அலுவலகங்களிலும் இப்படித்தான் கதைகள் நடக்குது.

    ஒரு நாள் ஒரு ஹார்ட்வேர் டெக்னீஷியனுடன் நடந்த உரையாடல், ஒரு சாமான்யமான வேலைக்கு எவ்வளவு பஞ்சாயத்து நடக்கும்னு காட்டும் வகையில் நம்மை சிரிக்கவும், சற்று சோகப்படவும் வைக்கிறது.

    SQL சர்வர் என்றால் என்ன? நம் ஊரு அலுவலகங்களில் வந்த குழப்பம்

    SQL என்றால் Structured Query Language. இது ஒரு தரவுத்தள மென்பொருள். வழக்கமாக, எதாவது மென்பொருள் நிறுவும் போது, அதுக்கு தரவுத்தள இணைப்பு தேவை. இல்லையென்றால் அது வேலை செய்யாது. ஆனா, நம்ம ஊரு அலுவலகங்களில், "SQL" என்றாலே அந்த சோதனையை Desktop-ல ஒரு ஐகான் காணோம் என்றால், "இதெல்லாம் வேலை செய்யலையே!" என்று பதறும் பழக்கம் இருக்கிறது.

    இந்த கதையில கூட, மென்பொருள் நிறுவ முடியாமல், வாடிக்கையாளரிடம் அனுமதிகள் சரியா உள்ளதா என்று கேட்கும்போது, ஹார்ட்வேர் டெக்னீஷியன் SQL சர்வர் Desktop-ல் எதுவும் காணலையேன்னு கவலைப்படுகிறார். "அது வேற சர்வர், வெப் சர்வருக்குள்ள அனுமதிகள் பத்தி பார்த்தீர்களா?" என்ற கேள்விக்கு பதில் கிடைக்கவே இல்லை!

    அனுபவத்தின் சிரிப்பு – உங்க அலுவலகத்தில் நடந்திருக்குமா?

    இந்த சம்பவம் படிச்சவுடன், நம்ம அலுவலகத்துல நடந்த சில அனுபவங்கள் நினைவுக்கு வருது. ஒருத்தர் சொல்றாங்க, "என் கூட்டாளி ஒருத்தர் என்னிடம் உதவி கேட்டப்போ, நான் சொன்னதை கேட்காம, வேறு ஏதோ செய்துட்டு, 'இன்னும் வேலை செய்யலையே!'ன்னு வருவாங்க. 5 நிமிஷம் வேலையை 30 நிமிஷத்துக்கு நீட்டி வைப்பாங்க!"

    இதே மாதிரி, மற்றொரு வாசகர் சொல்றார், "எப்போதாவது பொறுமை தீர்ந்துவிடும். இரண்டு முறை சொல்லி விட்டதும், 'நீங்க அந்த விஷயத்தை செஞ்சதும், ஸ்கிரீன் ஷாட் எடுத்ததும், எனக்கு ஈமெயில் அனுப்புங்க'ன்னு சொல்லி போயிடுவேன். ஆனா அந்த ஒரு நாளில், அது தான் சரியான தீர்வு இல்லாத நாளாயிடும்!"

    இந்த மாதிரி சம்பந்தப்பட்ட அனுபவங்கள், நம்ம ஊரு அலுவலக கலாச்சாரத்திலேயும் நிறைய. Whatsapp-ல் பதில் சொல்லி, அதே கேள்வி மீண்டும் மீண்டும் வரும் சந்தர்ப்பம் எத்தனை பேருக்கு தெரியாதது!

    அனுமதிகள், SQL, மற்றும் பயங்கரமான குழப்பங்கள்

    ஒரு வாசகர் சொல்லியிருக்கிறார், "நம்ம ஆட்கள் SQL, அனுமதிகள், வெப் பக்கம், எல்லாத்தையும் ஒன்றாகக் குழப்பிவிடுவாங்க. ஒரு பக்கம் அனுமதிகள் சரியா இல்லைன்னு சொன்னா, SQL சர்வரை uninstall பண்ணலாமா என்று கேட்பாங்க!"

    அடுத்தவர், "அப்பா, என்னிடம் உதவி கேட்கும்போது, 'OK' பொத்தானை அழுத்து'ன்னு சொன்னேன். 'CANCEL' பொத்தான் தெரியுது, அதை அழுத்தலாமா?'ன்னு கேட்கிறார். சொல்லி வைத்த இடத்தில், 'CANCEL' அழுத்தி விடுவார்!"

    இவை எல்லாம் நம்ம ஊரு அலுவலகங்களில் தினமும் நடக்குற காமெடி. பலர், புது மென்பொருள் வந்தாலே பயப்படுவாங்க. SQLன்னா, 'என்னமோ பெரிய விஷயம்!' என்று நினைப்பார்கள்.

    டெக் சப்போர்ட் வாழ்க்கையின் சிரிப்பு சோகங்கள்

    இந்த மாதிரி அனுபவங்கள், டெக் சப்போர்ட் துறையிலுள்ளவர்களுக்கு தினசரி சந்திக்க வேண்டிய சவால்கள். ஒரு வாசகர் ரசித்துப் பேசுறார், "நம்ம பணி ஒருவேளை மற்றவரின் கை, கண் ஆகி வேலை செய்வதாகவும், அவர்கள் சொல்லும் பதில்களை சந்தித்து, பொறுமையோடு விளக்கிக் கொண்டே இருக்க வேண்டுமாகவும் இருக்கிறது!"

    மற்றொரு வாசகர், "இப்படி SQL சர்வர் Desktop-ல தெரியலையேன்னு பதறுறவர்களுக்கு, 'Services'ல போய் SQL Server ஓடுதா பாருங்க'ன்னு சொன்னால் போதும்," என்று சொல்லி இருக்கிறார்.

    இவை எல்லாம் நம்ம ஊரு அலுவலக கலாச்சாரத்தில் SQL, அனுமதிகள், டெக் சப்போர்ட் – இவை பற்றிய புரிதல் எவ்வளவு முக்கியம் என்று நம்மை நினைவுபடுத்துகிறது.

    முடிவுரை – உங்களுக்கும் இப்படியொரு அனுபவம் இருக்கா?

    இந்தக் கதையைப் படித்து சிரிக்கவும், சற்று யோசிக்கவும் உங்களுக்கு நேர்ந்திருக்கும். உங்கள் அலுவலகத்தில், "SQL" என்றாலே பயம், அனுமதிகள் என்றாலே குழப்பம், மென்பொருள் என்றாலே தடுமாற்றம் – இவை எல்லாம் நடந்திருக்குமா? உங்கள் அனுபவங்களை கீழே கமெண்டில் பகிர்ந்து, நம்ம ஊரு அலுவலக கலாச்சார சிரிப்புக்கு இன்னும் நிறைய உப்பு சேர்ப்போம்!

    SQL சர்வருக்கும், நேர்த்தியான அனுமதிகளுக்கும் – வாழ்க நம் டெக் சப்போர்ட் போராளிகள்!

    அசல் ரெடிட் பதிவு: Won't somebody please think of the SQL server!
