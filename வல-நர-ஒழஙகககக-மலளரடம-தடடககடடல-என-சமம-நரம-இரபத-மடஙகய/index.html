    வேலை நேர ஒழுங்குக்காக மேலாளரிடம் தட்டிக்கேட்டால்... என் சும்மா நேரம் இருபது மடங்காய்!

    "காலை 9 மணிக்கு அலுவலகம், பிறகு எப்போதும் போல் வேலை..." – இப்படி ஒரு சாதாரண நாள் தான் எனக்கும். ஆனா ஒரு நாள் மேலாளர் என்னை, "நீ அட்டவணையை (rota) கண்டிப்பா பின்பற்றணும்!" என்று தட்டிக்கேட்டார். என்னோட 'மாலிசியஸ் கம்ப்ளையன்ஸ்' (Malicious Compliance) என்ன நடந்துச்சு தெரியுமா? இதோ, உங்க முன்னாடி அந்த அற்புதமான அலுவலக சூழல் கதையை சொல்றேன்.

    இது என் கால் சென்டர் (Call Centre) அனுபவம். சில வருடங்களுக்கு முன்னாடி ஒருத்தர் விட்டு போனதால, அவருடைய மேலாண்மை (admin) பொறுப்பும் எனக்கு மேலே மிதந்து வந்துச்சு. இது வேலைக்கு சேரும் நேரத்துக்கும், பிற்பகல் ஓய்வு நேரத்துக்கும் இடையே பத்து, பதினைந்து நிமிஷம் தான் ஆகும் சிறிய பணிகள். ஆனா, மேனேஜர் சொன்னபடி ரோட்டாவை பின்பற்றணும் என்கிற கட்டாயம் வந்ததும், என் வாழ்க்கையே வித்தியாசமாயிற்று!

    "ரோட்டா" என்றால் என்ன? தமிழ் அலுவலகங்களில் அது எப்படி?

    நம்ம ஊரிலே பலருக்கு 'ரோட்டா' (rota) என்றால் புதுசு தான். இது ஒரு அலுவலக அட்டவணை மாதிரி – உங்களுக்கு எந்த நேரத்தில் என்ன வேலை செய்யணும் என்று லிஸ்ட் பண்ணி வைக்கும். இரவு ரோட்டா, பகல் ரோட்டா, வேறு வேலை ரோட்டா – இதெல்லாம் நம்ம பஸ்கள் போலவே! என் மேலாளர், "காலை 15 நிமிஷம், பிற்பகல் 15 நிமிஷம் admin வேலை செய்யணும்; அந்த நேரம் admin code-ல் இருக்கணும்," என்று கட்டளை போட்டார்.

    ஆனா, நம்ம ஊரு பழக்கப்படி, வேலை இருக்கும்னு பார்த்து இடையில் admin வேலை முடிச்சுடுவோம். அதான் நீண்ட நாள் வழக்கம்.

    மேலாளரின் கட்டளை – "சரியா பின்பற்றணும்!"

    நான் முதலில், "நம்ப ரோட்டா போன நேரம் admin வேலை செய்யாதா, வேற நேரம் முடிச்சா என்ன பிரச்சினை?" என்று பதிலளிக்க நினைத்தேன். ஆனா மேலாளரின் கட்டளை கடுமை! "நீ admin code-க்கு போகணும்; அந்த நேரத்துல தான் admin வேலை செய்யணும்!"

    நான் என்ன பண்ணினேன் தெரியுமா? வேலை நேரத்தைத் தவிர, காலை 8:55-க்கும், இடைவேளைகளிலும் admin வேலை முடிச்சு, 9 மணிக்கு admin code-க்கு போய் உட்கார்ந்தேன். வேலை ஏற்கனவே முடிஞ்சிருச்சு; அந்த 15 நிமிஷம் நான் சும்மா YouTube-ல் வீடியோ பார்த்தேன், WhatsApp-ல் அழகான memes பார்த்தேன்!

    பிற்பகலும் அதே போலவே – admin வேலை ஏற்கனவே முடிஞ்சிருச்சு; admin code-க்கு போய், சும்மா தண்ணி குடிச்சு, சற்றே ஓய்வெடுத்து, சீரியல் சுவாரஸ்யம் பார்த்தேன். என் வேலைநேரம், நேர்த்தியான சும்மா நேரமா மாறியது!

    இந்த "மாலிசியஸ் கம்ப்ளையன்ஸ்" புரிகிறதா? சமூக வலைத்தள விமர்சனங்கள்

    இந்த கதையை Reddit-ல் பகிர்ந்ததும், பல மக்கள் என்ன சொன்னார்கள் தெரியுமா? "நீ செய்வது compliance-யும் இல்லை, malicious-யும் இல்லை!" என்று ஒருவர் விமர்சித்தார். இன்னொருவர், "நீ வேலை நேரத்துக்கு வெளியே வேலை செய்து, உன் சும்மா நேரத்தை வேறு நேரத்துக்கு மாற்றுறியே! இது வெற்றி என்று நினைக்குறது உனக்கு மட்டுமே," என்று கலாய்த்தார்.

    மற்றொருவர், "சாமி, நீ தண்ணி குடிக்கும்போதும், பல் துலக்கும்போதும் ஒரே நேரத்தில் இரண்டையும் முடிப்பது போல brag பண்ணுறாய்!" என்று நம்ம ஊரு கலாட்டா டைலிலே சொன்னாரு.

    ஒரு மூத்த கமெண்ட், "மேலாளர் சொன்ன நாட்கள் admin வேலை குறைவாக நடக்கிறது என்று மேலதிகாரிகள் கவனிக்க உதவும். நீ admin வேலை நேரம் சரியாக பின்பற்றினாலே, உனக்கு கூடுதல் சம்பளம் கேட்க வாய்ப்பு வரும்," என்று அறிவுரை சொல்லி இருந்தார்.

    இதிலேயே இன்னொரு சுவாரஸ்யம் – பலர், "நீ வேலை நேரத்துக்கு வெளியே admin வேலை செய்து, அந்நியாயம் செய்து கொண்டிருப்பதை யாரும் பாராட்ட மாட்டார்கள்; இது உனக்கு ஹானி மட்டும் தான்," என்று எச்சரித்தனர்.

    நம்ம ஊரு அலுவலக கலாச்சாரத்துக்கு இது எப்படி பொருந்தும்?

    இது நம்ம ஊரு அலுவலகங்களில் மிகப் பொதுவான பிரச்சினை – ஒரு சில பேர், மேலாளர் சொன்னப்படி நேரத்தை மதிக்காமல், வேலை நேரத்துக்கு வெளியே கூட ஒதுக்கி வேலையை முடிக்க நினைப்பார்கள். "வேலை முடிஞ்சா போதும்; நேரம் பார்த்து, code பார்த்து என்ன பயன்?" என்று நினைப்பார்கள்.

    ஆனா, இது நம்ம பலருக்கும் தீங்கு பயக்கும். நீ admin வேலை நேரம் பின்பற்றாமல் முன்னமே முடிச்சா, மேலாளருக்கு admin நேரம் குறைவாகவே தெரியும். அப்புறம் அந்த நேரம் வெறும் சும்மா நேரம் ஆகி, புது வேலைகள் வந்து சேரும். மற்றவர்களும், "அவன் போன நேரத்தில் admin வேலை முடிச்சான், நீ ஏன் முடியவில்லை?" என்று மேலாளர் கேட்பார்.

    இதில் நம்ம பழமொழி சரியானது – "கொஞ்சம் கூட நேரம் வீணாக்காதே; உன் நேரத்தை மதித்துக் கொள்!"

    நம்மால் என்ன கற்க முடியும்?

    இந்த கதையில் இருந்து நம்மால் கற்றுக்கொள்ள வேண்டியது – மேலாளரின் கட்டளையை நேர்த்தியாக, நேரத்துக்குள், வேலை நேரத்திலேயே பின்பற்றுங்கள். உங்களுக்கான admin நேரம் அட்டவணையில் இருந்தால், அப்படியே பின்பற்றுங்கள். வேலை நேரத்துக்கு வெளியே கூடுதல் வேலை செய்து, சும்மா நேரத்தை மாற்றாதீர்கள். உங்க நேரம், உங்க மதிப்பு.

    இப்படி ஒரு நேரம் கிடைத்தால் மட்டும், அது உங்களுக்கான ஓய்வு நேரமா, இல்லை மேலாளர் உங்களை மீண்டும் வேறு வேலைக்கு அழைப்பாரா என்பது பெரிய கேள்வி தான்!

    முடிவில்...

    நம்ம ஊரு அலுவலகத்தில் சக ஊழியர்களுடன் நேரத்தைப் பகிர்ந்துகொள்; மேலாளரிடம் நேரத்தை மதிப்பது எப்படி என்று சொல்லுங்கள். "ரோட்டா" அட்டவணையை பின்பற்றுவது, நீங்களும் உங்க குழுவும் நல்ல எதிர்காலம் பெற உதவும். உங்க அலுவலக அனுபவங்களும், சும்மா நேர கலாட்டாக்களும் கீழே கமெண்டில் பகிருங்கள். உங்கள் அலுவலக வாழ்க்கை எப்படி இருக்கிறது, எங்களைப் போல உங்களுக்கும் "மாலிசியஸ் கம்ப்ளையன்ஸ்" ஏதேனும் நடந்ததா?

    நம் பக்கத்துல வேலை செய்தாலும், நேரத்தை மதிப்பது தான் பெரிய விஷயம். உங்கள் நண்பர், சக ஊழியர்கள், மேலாளர்கள் – எல்லாரும் இதை நினைவில் வைக்கணும்!

    உங்க அலுவலக கலாட்டா அனுபவம் கீழே கண்டிப்பா பகிருங்க!
